## File Naming Conventions

| Type | Convention | Example |
|------|------------|---------|
| Components | PascalCase | `UserCard.tsx`, `UserCard.vue` |
| Hooks | camelCase with use | `useAuth.ts` |
| Composables | camelCase with use | `useAuth.ts` |
| Utilities | camelCase | `formatDate.ts` |
| Types | camelCase with .types | `user.types.ts` |
| Tests | .test or .spec | `UserCard.test.tsx` |
| Styles | .module.css | `Button.module.css` |
| API Routes | route.ts | `app/api/users/route.ts` |

## Import Conventions

```typescript
// 1. External packages (alphabetical)
import { useState } from 'react'
import { z } from 'zod'

// 2. Internal aliases (@ prefix)
import { Button } from '@/components/ui'
import { useAuth } from '@/hooks/useAuth'

// 3. Relative imports (local files)
import { helper } from './utils'

// 4. Types (type imports)
import type { User } from '@/types'
```
