## Components

### Buttons
| Variant | Use Case | Style |
|---------|----------|-------|
| Primary | Main actions | Solid background, white text |
| Secondary | Secondary actions | Outlined, primary color |
| Ghost | Tertiary actions | No background, text only |
| Destructive | Dangerous actions | Red background |

Sizes: `sm` (32px), `md` (40px), `lg` (48px)

### Forms
| Component | Description |
|-----------|-------------|
| Input | Text input with label and error state |
| Select | Dropdown with search support |
| Checkbox | Multi-select options |
| Radio | Single-select options |
| Switch | Toggle on/off |
| Textarea | Multi-line text input |

Validation states: default, focus, error, disabled

### Modals/Dialogs
- Center aligned
- Backdrop blur
- Close on escape key
- Trap focus within modal
- Sizes: sm (400px), md (500px), lg (600px), full

### Navigation
- Header: Logo, nav links, user menu
- Sidebar: Collapsible, nested items
- Breadcrumbs: Current location path
- Tabs: Content switching

## Animation

### Transitions
```css
--duration-fast:   150ms;
--duration-normal: 200ms;
--duration-slow:   300ms;

--easing-default:  cubic-bezier(0.4, 0, 0.2, 1);
--easing-in:       cubic-bezier(0.4, 0, 1, 1);
--easing-out:      cubic-bezier(0, 0, 0.2, 1);
```

### Motion Principles
- Use motion to provide feedback
- Keep animations subtle and purposeful
- Respect reduced-motion preferences
- Avoid animations longer than 300ms

## Accessibility

- Color contrast: WCAG AA minimum (4.5:1 text, 3:1 UI)
- Focus indicators: Visible ring on all interactive elements
- Screen reader support: Proper ARIA labels and roles
- Keyboard navigation: All features accessible via keyboard
- Touch targets: Minimum 44x44px for mobile
